# 考研英语完形填空高频考点

## 一、完形填空题型总览

考研英语完形填空共20题，每题0.5分，总计10分。文章约240-280词，设有20个空白，每个空白提供四个选项。完形填空考查的是考生的综合语言运用能力，包括词汇辨析、固定搭配、逻辑关系和语法知识等多个维度。

完形填空的核心解题原则是"上下文语境决定答案"。每个空白的正确选项都不是孤立存在的，而是由文章的整体逻辑和局部语境共同决定的。因此，解题时必须始终关注上下文的语义连贯性和逻辑一致性。

---

## 二、逻辑关系词考点

逻辑关系词是完形填空最重要的考点之一，每年必考3-5题。准确识别上下文的逻辑关系是选择正确衔接词的关键。

### 2.1 转折关系

转折关系表示前后文语义相反或相对，是最常考的逻辑关系类型。

#### 2.1.1 核心转折词辨析

| 转折词 | 用法与区别 |
|-------|----------|
| however | 最常用的转折词，可位于句首、句中或句末，通常用逗号隔开。语气较为正式，强调前后对比 |
| but | 最基本的转折连词，直接连接两个分句，语气比however更强烈、更口语化 |
| yet | 语气较but弱，含有"尽管如此"的意味，常与and连用为and yet |
| nevertheless | 正式用语的转折词，语气比however更强，强调尽管有前文情况，后文仍然成立 |
| nonetheless | 与nevertheless基本同义，但使用频率较低，语气更为正式 |
| whereas | 引导对比从句，强调两者的差异，通常连接两个对比性的事实 |
| while | 可表示转折"虽然"，也可表示对比"然而"，需根据语境判断 |
| although / though | 引导让步状语从句，注意主句中不能再用but |
| on the contrary | 表示"恰恰相反"，用于否定前文观点并给出相反事实 |
| on the other hand | 表示"另一方面"，用于引出不同角度的观点 |
| in contrast | 表示"相比之下"，用于两者之间的对比 |

#### 2.1.2 转折关系解题要点

1. 转折词前后语义必须相反或相对，选择转折词前需确认前后文确实存在语义转折
2. 区分强转折（but、however）和弱转折（although、though），根据语境选择合适语气的词
3. 注意however的位置灵活性，它可出现在句首、句中或句末

### 2.2 因果关系

因果关系表示前后文之间存在原因与结果的逻辑联系。

#### 2.2.1 核心因果词辨析

| 因果词 | 用法与区别 |
|-------|----------|
| because | 最基本的因果连词，引导原因从句，语气直接 |
| since / as | 语气比because弱，通常表示已知原因，侧重结果 |
| for | 并列连词，引导的原因是补充说明性的，语气最弱 |
| therefore | 副词，表示"因此"，用于引出结果，通常位于句首或句中 |
| consequently | 正式用语，表示"因此、所以"，强调因果的必然性 |
| thus | 表示"因此、从而"，可引出结果，也可表示方式 |
| hence | 正式用语，表示"因此"，常用于推理结论 |
| so | 最基本的因果副词，语气较口语化 |
| as a result | 短语，表示"结果是"，强调因果关系 |
| result in | 动词短语，表示"导致"，主语是原因，宾语是结果 |
| result from | 动词短语，表示"起因于"，主语是结果，宾语是原因 |
| due to / owing to | 介词短语，表示"由于"，后接名词或名词短语 |
| contribute to | 动词短语，表示"促成、导致"，强调多因素中某一因素的作用 |

#### 2.2.2 因果关系解题要点

1. 明确因果方向：判断空白处需要的是表示原因的词还是表示结果的词
2. 区分原因的强弱：because语气最强，for语气最弱，根据语境选择
3. 注意result in与result from的方向差异，这是常考的混淆点

### 2.3 递进关系

递进关系表示后文在前文基础上进一步深入或扩展。

#### 2.3.1 核心递进词辨析

| 递进词 | 用法与区别 |
|-------|----------|
| moreover | 正式用语，表示"此外、而且"，用于补充新的论据 |
| furthermore | 与moreover基本同义，但更强调递进层次，语气更正式 |
| besides | 表示"此外"，带有"除了……还有"的意味 |
| in addition | 短语，表示"此外、另外"，用于补充信息 |
| what is more | 表示"更重要的是"，强调递进内容的分量 |
| additionally | 副词，表示"此外"，语气中性 |
| even | 表示"甚至"，用于强调递进的意外性 |
| indeed | 表示"确实、的确"，用于确认并加强前文内容 |

#### 2.3.2 递进关系解题要点

1. 递进词前后语义方向一致，但后文内容比前文更进一步
2. 区分moreover和furthermore的正式程度差异
3. even表示的递进具有意外性，前后文应有"出乎意料"的逻辑

### 2.4 并列关系

并列关系表示前后文语义平行、地位等同。

#### 2.4.1 核心并列词辨析

| 并列词 | 用法与区别 |
|-------|----------|
| and | 最基本的并列连词，连接并列的词、短语或句子 |
| or | 表示选择关系的并列，"或者" |
| nor | 用于否定句的并列，"也不" |
| similarly | 副词，表示"类似地"，用于类比并列 |
| likewise | 副词，表示"同样地"，用于类比并列 |
| equally | 副词，表示"同样地、相等地" |
| as well as | 连接并列成分，强调前者 |
| not only...but also | 表示"不仅……而且"，强调递进式并列 |

## 四、词义辨析考点

词义辨析是完形填空中难度最高的题型，要求考生在四个近义词或形近词中选择最符合语境的选项。

### 4.1 高频近义词辨析

#### 4.1.1 表示"影响"的词

- affect：动词，表示"影响"，侧重影响的动作过程
- effect：名词表示"影响、效果"，动词表示"实现、引起"
- influence：动词和名词，表示"影响"，侧重潜移默化的影响
- impact：名词和动词，表示"冲击、影响"，强调影响的力度
- interfere：动词，表示"干涉"，带有负面含义

#### 4.1.2 表示"观察"的词

- observe：仔细观察，也指遵守规则
- notice：注意到，偶然发现
- watch：注视、观看，强调持续关注
- perceive：感知、察觉，侧重心理过程

#### 4.1.3 表示"要求"的词

- require：客观需要，正式要求
- demand：强烈要求，语气较重
- request：礼貌请求，语气较委婉
- claim：声称有权要求

#### 4.1.4 表示"限制"的词

- limit：设定界限，限制数量或程度
- restrict：限制活动范围或自由
- confine：限制在某一范围内
- constrain：约束、压制，侧重外力限制

### 4.2 形近词辨析

| 词组 | 区别 |
|------|------|
| adapt / adopt / adept | adapt适应 / adopt采纳 / adept熟练的 |
| affect / effect | affect影响(动词) / effect效果(名词) |
| arise / rise / raise | arise出现(不及物) / rise上升(不及物) / raise举起(及物) |
| assure / ensure / insure | assure使确信 / ensure确保 / insure保险 |
| complement / compliment | complement补充 / compliment赞美 |
| principal / principle | principal主要的 / principle原则 |
| stationary / stationery | stationary静止的 / stationery文具 |
| considerate / considerable | considerate体贴的 / considerable相当大的 |

## 五、语法考点

### 5.1 虚拟语气

虚拟语气是完形填空的常考语法点，需掌握以下形式：

- 与现在事实相反：If + did/were, would/should/could/might + do
- 与过去事实相反：If + had done, would/should/could/might + have done
- 与将来事实相反：If + were to do / should do, would/should/could/might + do
- 省略if的倒装：Had I known... / Were I... / Should you...
- suggest/demand/require等词后的that从句用(should) + 动词原形

### 5.2 非谓语动词

- 不定式：表示目的、将来或具体动作
- 动名词：表示习惯性、一般性动作，作主语或宾语
- 分词：现在分词表示主动和进行，过去分词表示被动和完成

### 5.3 代词指代

完形填空中代词指代是重要考点，需根据上下文确定代词的指代对象：

- it：指代前文出现的单数名词或不可数名词
- they/them：指代前文出现的复数名词
- this/that：可指代前文的句子内容
- such：指代前文提到的同类事物

## 六、完形填空解题策略

### 6.1 首句必读原则

完形填空的首句通常不设空，是理解文章主题和基调的关键入口。首句往往点明文章的话题、观点或背景，为后续解题提供方向。

### 6.2 语境推断法

根据空白处前后的语义和逻辑关系推断正确答案。具体步骤：

1. 通读全文，把握文章大意和情感基调
2. 分析空白处前后的逻辑关系
3. 根据逻辑关系选择对应的衔接词或语义方向
4. 将选项代入验证，确保语义通顺、逻辑连贯

### 6.3 排除法

当无法直接确定答案时，使用排除法逐步缩小范围：

1. 排除与上下文逻辑矛盾的选项
2. 排除搭配不当的选项
3. 排除语义重复的选项
4. 在剩余选项中选择最符合语境的答案

### 6.4 复现法

文章中前后出现的同义复现、反义复现和上下义复现是解题的重要线索。如果空白处与上下文存在词汇复现关系，正确选项往往与复现词构成同义或反义关系。

## 七、备考建议

1. **逻辑词专项训练**：系统整理各类逻辑关系词，通过专项练习强化辨析能力
2. **固定搭配积累**：每天记忆10-15个高频搭配，注意近义搭配的细微区别
3. **词义辨析训练**：通过近义词和形近词的对比练习，提高辨析准确率
4. **限时训练**：完形填空控制在15分钟内完成，培养快速判断能力
5. **真题精研**：反复研究历年真题，总结高频考点和出题规律
6. **错题归类**：将错题按考点分类整理，针对薄弱环节进行强化训练
