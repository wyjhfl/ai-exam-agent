# 线性代数重点知识

## 一、行列式

### 1.1 行列式的定义与性质

$n$ 阶行列式定义为：

$$|A| = \sum_{j_1 j_2 \cdots j_n} (-1)^{\tau(j_1 j_2 \cdots j_n)} a_{1j_1} a_{2j_2} \cdots a_{nj_n}$$

其中 $\tau(j_1 j_2 \cdots j_n)$ 为排列 $j_1 j_2 \cdots j_n$ 的逆序数。

**基本性质**：
- 行列式与它的转置行列式相等，即 $|A| = |A^T|$
- 互换行列式的两行（列），行列式变号
- 行列式的某一行（列）的公因子可以提到行列式外面
- 若行列式某一行（列）的元素都是两数之和，则行列式可拆成两个行列式之和
- 把行列式的某一行（列）的各元素乘以同一数加到另一行（列）对应元素上，行列式不变

### 1.2 行列式按行（列）展开

$$|A| = a_{i1}A_{i1} + a_{i2}A_{i2} + \cdots + a_{in}A_{in}$$

其中 $A_{ij} = (-1)^{i+j}M_{ij}$ 为 $a_{ij}$ 的代数余子式，$M_{ij}$ 为余子式。

**异行（列）展开为零**：$a_{i1}A_{j1} + a_{i2}A_{j2} + \cdots + a_{in}A_{jn} = 0$（$i \neq j$）

**重要公式**：
- 上（下）三角行列式等于主对角线元素之积
- 范德蒙德行列式：$|V_n| = \prod_{1 \leq j < i \leq n}(x_i - x_j)$
- $|kA| = k^n|A|$（$n$ 阶方阵）
- $|AB| = |A||B|$
- $|A^{-1}| = |A|^{-1}$
- $|A^*| = |A|^{n-1}$

**考研要点**：行列式计算常用方法有化三角法、展开降阶法、递推法、数学归纳法。抽象行列式计算常用 $|AB| = |A||B|$ 及 $|A^*| = |A|^{n-1}$。

## 二、矩阵运算

### 2.1 矩阵乘法性质

- 结合律：$(AB)C = A(BC)$
- 分配律：$A(B+C) = AB + AC$，$(B+C)A = BA + CA$
- 数乘：$\lambda(AB) = (\lambda A)B = A(\lambda B)$
- 转置：$(AB)^T = B^T A^T$
- **注意**：矩阵乘法一般不满足交换律，即 $AB \neq BA$；$AB = 0$ 不能推出 $A = 0$ 或 $B = 0$

### 2.2 逆矩阵

方阵 $A$ 可逆的充要条件是 $|A| \neq 0$，此时：

$$A^{-1} = \frac{1}{|A|}A^*$$

**求逆方法**：
- 伴随矩阵法：$A^{-1} = \frac{1}{|A|}A^*$
- 初等变换法：$(A|E) \xrightarrow{\text{初等行变换}} (E|A^{-1})$
- 分块矩阵求逆

**逆矩阵性质**：
- $(A^{-1})^{-1} = A$
- $(AB)^{-1} = B^{-1}A^{-1}$
- $(A^T)^{-1} = (A^{-1})^T$
- $(kA)^{-1} = \frac{1}{k}A^{-1}$（$k \neq 0$）

### 2.3 伴随矩阵

$$A A^* = A^* A = |A|E$$

由此可推出：$A^* = |A|A^{-1}$（$A$ 可逆时）

**伴随矩阵的重要性质**：
- $(A^*)^* = |A|^{n-2}A$（$n \geq 2$）
- $(AB)^* = B^* A^*$
- $(kA)^* = k^{n-1}A^*$
- $r(A^*)$ 与 $r(A)$ 的关系：若 $r(A) = n$，则 $r(A^*) = n$；若 $r(A) = n-1$，则 $r(A^*) = 1$；若 $r(A) < n-1$，则 $r(A^*) = 0$

### 2.4 矩阵的秩

矩阵 $A$ 的秩 $r(A)$ 是 $A$ 中最高阶非零子式的阶数。

**秩的重要性质**：
- $r(A) = r(A^T)$
- $r(AB) \leq \min\{r(A), r(B)\}$
- $r(A + B) \leq r(A) + r(B)$
- 若 $A$ 为 $m \times n$ 矩阵，$P$ 为 $m$ 阶可逆矩阵，$Q$ 为 $n$ 阶可逆矩阵，则 $r(PAQ) = r(A)$
- $r(A) + r(B) - n \leq r(AB)$（$A$ 为 $m \times n$，$B$ 为 $n \times s$）
- 若 $AB = 0$，则 $r(A) + r(B) \leq n$

**考研要点**：矩阵秩的问题是考研线性代数的核心考点，常与方程组、特征值结合出题。$AB = 0$ 时 $r(A) + r(B) \leq n$ 是高频考点。

## 三、向量组

### 3.1 线性相关与线性无关

设 $\alpha_1, \alpha_2, \cdots, \alpha_m$ 为 $n$ 维向量，若存在不全为零的数 $k_1, k_2, \cdots, k_m$，使得 $k_1\alpha_1 + k_2\alpha_2 + \cdots + k_m\alpha_m = 0$，则称向量组线性相关；否则称线性无关。

**判定方法**：
- 向量组 $\alpha_1, \alpha_2, \cdots, \alpha_m$ 线性相关 $\Leftrightarrow$ $r(\alpha_1, \alpha_2, \cdots, \alpha_m) < m$
- 向量组 $\alpha_1, \alpha_2, \cdots, \alpha_m$ 线性无关 $\Leftrightarrow$ $r(\alpha_1, \alpha_2, \cdots, \alpha_m) = m$
- $n$ 个 $n$ 维向量线性相关 $\Leftrightarrow$ 它们构成的行列式为零

**重要结论**：
- 含零向量的向量组必线性相关
- 两个向量线性相关 $\Leftrightarrow$ 它们成比例
- 部分组线性相关 $\Rightarrow$ 整体组线性相关；整体组线性无关 $\Rightarrow$ 部分组线性无关
- 若向量组中向量个数大于向量维数，则向量组必线性相关
- $\alpha_1, \alpha_2, \cdots, \alpha_m$ 线性无关，而 $\alpha_1, \alpha_2, \cdots, \alpha_m, \beta$ 线性相关，则 $\beta$ 可由 $\alpha_1, \alpha_2, \cdots, \alpha_m$ 线性表示，且表示法唯一

### 3.2 极大线性无关组与向量组的秩

向量组的极大线性无关组所含向量个数称为向量组的秩。

**矩阵的秩等于其行向量组的秩，也等于其列向量组的秩。**

## 四、线性方程组

### 4.1 齐次线性方程组 $Ax = 0$

设 $A$ 为 $m \times n$ 矩阵：
- $Ax = 0$ 有非零解 $\Leftrightarrow$ $r(A) < n$
- $Ax = 0$ 只有零解 $\Leftrightarrow$ $r(A) = n$
- 当 $m < n$ 时，$Ax = 0$ 必有非零解
- 当 $m = n$ 时，$Ax = 0$ 有非零解 $\Leftrightarrow$ $|A| = 0$

**解的结构**：设 $\eta_1, \eta_2, \cdots, \eta_{n-r}$ 为基础解系，则通解为 $x = k_1\eta_1 + k_2\eta_2 + \cdots + k_{n-r}\eta_{n-r}$，其中 $r = r(A)$。

**基础解系含 $n - r(A)$ 个解向量。**

### 4.2 非齐次线性方程组 $Ax = b$

**解的判定**：
- $Ax = b$ 有解 $\Leftrightarrow$ $r(A) = r(A|b)$
- $r(A) = r(A|b) = n$ 时，有唯一解
- $r(A) = r(A|b) < n$ 时，有无穷多解
- $r(A) < r(A|b)$ 时，无解

**解的结构**：设 $\eta^*$ 为 $Ax = b$ 的一个特解，$\eta_1, \eta_2, \cdots, \eta_{n-r}$ 为对应齐次方程组的基础解系，则通解为：

$$x = \eta^* + k_1\eta_1 + k_2\eta_2 + \cdots + k_{n-r}\eta_{n-r}$$

**解的性质**：
- 若 $\eta_1, \eta_2$ 是 $Ax = b$ 的解，则 $\eta_1 - \eta_2$ 是 $Ax = 0$ 的解
- 若 $\eta^*$ 是 $Ax = b$ 的解，$\xi$ 是 $Ax = 0$ 的解，则 $\eta^* + \xi$ 是 $Ax = b$ 的解

**考研要点**：方程组解的判定与解的结构是必考内容，常与矩阵的秩、向量组的线性表示结合出综合题。

## 五、特征值与特征向量

### 5.1 定义与求法

设 $A$ 为 $n$ 阶方阵，若存在数 $\lambda$ 和非零向量 $\alpha$，使得 $A\alpha = \lambda\alpha$，则 $\lambda$ 为 $A$ 的特征值，$\alpha$ 为对应于 $\lambda$ 的特征向量。

求法：解特征方程 $|\lambda E - A| = 0$ 得特征值，对每个特征值 $\lambda_i$，解 $(\lambda_i E - A)x = 0$ 得特征向量。

### 5.2 特征值的重要性质

- $|A| = \lambda_1 \lambda_2 \cdots \lambda_n$（特征值之积等于行列式）
- $\text{tr}(A) = \lambda_1 + \lambda_2 + \cdots + \lambda_n$（特征值之和等于迹）
- $A$ 可逆 $\Leftrightarrow$ $A$ 的特征值均不为零
- 不同特征值对应的特征向量线性无关
- 实对称矩阵的不同特征值对应的特征向量正交
- 实对称矩阵的特征值均为实数

**特征值的运算性质**：

| $A$ 的特征值 | $\lambda$ | $kA$ | $A^k$ | $f(A)$ | $A^{-1}$ | $A^*$ |
|---|---|---|---|---|---|---|
| 对应特征值 | $\lambda$ | $k\lambda$ | $\lambda^k$ | $f(\lambda)$ | $\lambda^{-1}$ | $\frac{|A|}{\lambda}$ |

### 5.3 相似矩阵

若存在可逆矩阵 $P$，使得 $P^{-1}AP = B$，则称 $A$ 与 $B$ 相似。

**相似矩阵的性质**：
- $|A| = |B|$
- $\text{tr}(A) = \text{tr}(B)$
- $r(A) = r(B)$
- $A$ 与 $B$ 有相同的特征值
- $A$ 与 $B$ 有相同的特征多项式

**矩阵可对角化的条件**：
- $A$ 可对角化 $\Leftrightarrow$ $A$ 有 $n$ 个线性无关的特征向量
- $A$ 有 $n$ 个不同的特征值 $\Rightarrow$ $A$ 可对角化
- 实对称矩阵一定可正交对角化

**考研要点**：特征值与特征向量是线性代数最重要的内容之一，常与二次型、矩阵对角化结合出大题。实对称矩阵的正交对角化是高频考点。

## 六、二次型

### 6.1 二次型的定义

$n$ 个变量的二次齐次多项式 $f(x_1, x_2, \cdots, x_n) = x^T A x$ 称为二次型，其中 $A$ 为实对称矩阵。

二次型的秩 = $r(A)$

### 6.2 正交变换化二次型为标准形

步骤：
1. 写出二次型的矩阵 $A$
2. 求出 $A$ 的特征值 $\lambda_1, \lambda_2, \cdots, \lambda_n$ 和对应的特征向量
3. 将特征向量正交化、单位化得正交矩阵 $Q$
4. 令 $x = Qy$，则 $f = \lambda_1 y_1^2 + \lambda_2 y_2^2 + \cdots + \lambda_n y_n^2$

### 6.3 正定二次型

二次型 $f = x^T A x$ 正定 $\Leftrightarrow$ 以下任一条件成立：
- $A$ 的所有特征值均大于零
- $A$ 的各阶顺序主子式均大于零
- $f$ 的正惯性指数为 $n$
- $A$ 合同于单位矩阵 $E$
- 存在可逆矩阵 $C$，使得 $A = C^T C$

**考研要点**：正交变换化二次型为标准形是必考大题，需熟练掌握。正定性的判定也是常考内容，顺序主子式法是判定正定的常用方法。配方法化二次型为标准形也需掌握，但正交变换法更重要。
