# 考研数学公式速查

## 一、高等数学

### 1. 重要极限
- lim(x→0) sin(x)/x = 1
- lim(x→∞) (1+1/x)^x = e
- lim(x→0) (1+x)^(1/x) = e

### 2. 导数公式
- (x^n)' = nx^(n-1)
- (sin x)' = cos x
- (cos x)' = -sin x
- (e^x)' = e^x
- (ln x)' = 1/x
- (a^x)' = a^x · ln a

### 3. 积分公式
- ∫x^n dx = x^(n+1)/(n+1) + C (n≠-1)
- ∫1/x dx = ln|x| + C
- ∫e^x dx = e^x + C
- ∫sin x dx = -cos x + C
- ∫cos x dx = sin x + C

### 4. 泰勒公式
- e^x = 1 + x + x²/2! + x³/3! + ...
- sin x = x - x³/3! + x⁵/5! - ...
- cos x = 1 - x²/2! + x⁴/4! - ...
- ln(1+x) = x - x²/2 + x³/3 - ... (-1 < x ≤ 1)

### 5. 中值定理
- 罗尔定理：f(a)=f(b)，则存在ξ∈(a,b)，使f'(ξ)=0
- 拉格朗日中值定理：存在ξ∈(a,b)，使f'(ξ)=[f(b)-f(a)]/(b-a)
- 柯西中值定理：f和g在[a,b]上满足条件，则存在ξ使f'(ξ)/g'(ξ)=[f(b)-f(a)]/[g(b)-g(a)]

## 二、线性代数

### 1. 行列式
- |kA| = k^n × |A|（n阶方阵）
- |AB| = |A|·|B|
- |A^T| = |A|
- |A⁻¹| = 1/|A|

### 2. 矩阵运算
- (AB)^T = B^T A^T
- (AB)⁻¹ = B⁻¹ A⁻¹
- r(AB) ≤ min{r(A), r(B)}

### 3. 特征值与特征向量
- Σλᵢ = tr(A) = Σaᵢᵢ
- Πλᵢ = |A|
- 不同特征值对应的特征向量线性无关
- 实对称矩阵的特征值都是实数，不同特征值对应的特征向量正交

## 三、概率论与数理统计

### 1. 常见分布
| 分布 | E(X) | D(X) |
|------|------|------|
| B(n,p) | np | np(1-p) |
| P(λ) | λ | λ |
| U(a,b) | (a+b)/2 | (b-a)²/12 |
| N(μ,σ²) | μ | σ² |
| Exp(λ) | 1/λ | 1/λ² |

### 2. 期望与方差性质
- E(aX+bY) = aE(X)+bE(Y)
- D(X) = E(X²) - [E(X)]²
- X,Y独立时：D(X+Y) = D(X)+D(Y)
- X,Y独立时：E(XY) = E(X)·E(Y)

### 3. 大数定律与中心极限定理
- 切比雪夫大数定律
- 伯努利大数定律
- 辛钦大数定律
- 列维-林德伯格中心极限定理（独立同分布）
- 棣莫弗-拉普拉斯中心极限定理（二项分布的正态近似）
