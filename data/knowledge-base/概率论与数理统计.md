# 概率论与数理统计

## 一、随机事件与概率

### 1.1 随机事件的关系与运算

- **包含**：$A \subset B$ 表示事件 $A$ 发生必导致事件 $B$ 发生
- **并（和）**：$A \cup B$ 表示事件 $A$ 与 $B$ 至少发生一个
- **交（积）**：$A \cap B$（或 $AB$）表示事件 $A$ 与 $B$ 同时发生
- **差**：$A - B = A\bar{B}$ 表示 $A$ 发生而 $B$ 不发生
- **互斥（互不相容）**：$AB = \emptyset$
- **对立（互补）**：$A \cup \bar{A} = \Omega$，$A\bar{A} = \emptyset$

**德摩根律**：$\overline{A \cup B} = \bar{A} \cap \bar{B}$，$\overline{A \cap B} = \bar{A} \cup \bar{B}$

### 1.2 概率的基本性质

- 非负性：$P(A) \geq 0$
- 规范性：$P(\Omega) = 1$
- 可列可加性：若 $A_1, A_2, \cdots$ 两两互斥，则 $P(\bigcup_{i=1}^{\infty} A_i) = \sum_{i=1}^{\infty} P(A_i)$

**推论**：
- $P(\bar{A}) = 1 - P(A)$
- $P(A - B) = P(A) - P(AB)$
- $P(A \cup B) = P(A) + P(B) - P(AB)$
- 若 $A \subset B$，则 $P(A) \leq P(B)$

### 1.3 条件概率

$$P(A|B) = \frac{P(AB)}{P(B)}, \quad P(B) > 0$$

**乘法公式**：$P(AB) = P(A)P(B|A) = P(B)P(A|B)$

推广：$P(A_1 A_2 \cdots A_n) = P(A_1)P(A_2|A_1)P(A_3|A_1 A_2)\cdots P(A_n|A_1 A_2 \cdots A_{n-1})$

### 1.4 全概率公式与贝叶斯公式

**全概率公式**：设 $B_1, B_2, \cdots, B_n$ 为样本空间的一个划分（完备事件组），则：

$$P(A) = \sum_{i=1}^{n} P(B_i)P(A|B_i)$$

**贝叶斯公式**：

$$P(B_j|A) = \frac{P(B_j)P(A|B_j)}{\sum_{i=1}^{n} P(B_i)P(A|B_i)}$$

**考研要点**：全概率公式和贝叶斯公式是必考内容，关键在于正确划分完备事件组。贝叶斯公式本质上是条件概率与全概率公式的结合。

### 1.5 事件的独立性

$A$ 与 $B$ 相互独立 $\Leftrightarrow$ $P(AB) = P(A)P(B)$

**注意**：
- 独立与互斥的关系：若 $P(A) > 0$ 且 $P(B) > 0$，则 $A$ 与 $B$ 独立和互斥不能同时成立
- $A$ 与 $B$ 独立 $\Leftrightarrow$ $A$ 与 $\bar{B}$ 独立 $\Leftrightarrow$ $\bar{A}$ 与 $B$ 独立 $\Leftrightarrow$ $\bar{A}$ 与 $\bar{B}$ 独立
- 三个事件相互独立需要两两独立且 $P(ABC) = P(A)P(B)P(C)$，两两独立推不出相互独立

## 二、一维随机变量及其分布

### 2.1 离散型随机变量

**常见离散分布**：

**（1）二项分布 $B(n,p)$**：

$$P(X = k) = C_n^k p^k (1-p)^{n-k}, \quad k = 0, 1, 2, \cdots, n$$

$E(X) = np$，$D(X) = np(1-p)$

**（2）泊松分布 $P(\lambda)$**：

$$P(X = k) = \frac{\lambda^k e^{-\lambda}}{k!}, \quad k = 0, 1, 2, \cdots$$

$E(X) = \lambda$，$D(X) = \lambda$

泊松定理：当 $n$ 很大、$p$ 很小时，$B(n,p) \approx P(np)$

**（3）几何分布 $G(p)$**：

$$P(X = k) = (1-p)^{k-1}p, \quad k = 1, 2, \cdots$$

$E(X) = \frac{1}{p}$，$D(X) = \frac{1-p}{p^2}$

### 2.2 连续型随机变量

**常见连续分布**：

**（1）均匀分布 $U(a,b)$**：

$$f(x) = \begin{cases} \frac{1}{b-a}, & a < x < b \\ 0, & \text{其他} \end{cases}$$

$E(X) = \frac{a+b}{2}$，$D(X) = \frac{(b-a)^2}{12}$

**（2）指数分布 $Exp(\lambda)$**：

$$f(x) = \begin{cases} \lambda e^{-\lambda x}, & x > 0 \\ 0, & x \leq 0 \end{cases}$$

$E(X) = \frac{1}{\lambda}$，$D(X) = \frac{1}{\lambda^2}$

指数分布的无记忆性：$P(X > s + t | X > s) = P(X > t)$

**（3）正态分布 $N(\mu, \sigma^2)$**：

$$f(x) = \frac{1}{\sqrt{2\pi}\sigma}e^{-\frac{(x-\mu)^2}{2\sigma^2}}, \quad -\infty < x < +\infty$$

$E(X) = \mu$，$D(X) = \sigma^2$

标准化：若 $X \sim N(\mu, \sigma^2)$，则 $Z = \frac{X - \mu}{\sigma} \sim N(0,1)$

$P(\mu - \sigma < X < \mu + \sigma) \approx 0.6826$

$P(\mu - 2\sigma < X < \mu + 2\sigma) \approx 0.9544$

$P(\mu - 3\sigma < X < \mu + 3\sigma) \approx 0.9974$

**考研要点**：正态分布是概率论中最重要的分布，其线性变换性质 $aX + b \sim N(a\mu + b, a^2\sigma^2)$ 常考。独立正态变量的线性组合仍为正态分布。

## 三、二维随机变量

### 3.1 联合分布与边缘分布

**离散型**：联合分布律 $P(X = x_i, Y = y_j) = p_{ij}$

边缘分布：$P(X = x_i) = \sum_j p_{ij}$，$P(Y = y_j) = \sum_i p_{ij}$

**连续型**：联合密度函数 $f(x,y)$

边缘密度：$f_X(x) = \int_{-\infty}^{+\infty} f(x,y)dy$，$f_Y(y) = \int_{-\infty}^{+\infty} f(x,y)dx$

### 3.2 条件分布

**离散型**：$P(Y = y_j | X = x_i) = \frac{p_{ij}}{p_{i\cdot}}$

**连续型**：$f_{Y|X}(y|x) = \frac{f(x,y)}{f_X(x)}$

### 3.3 随机变量的独立性

$X$ 与 $Y$ 相互独立 $\Leftrightarrow$ $F(x,y) = F_X(x)F_Y(y)$

离散型等价条件：$p_{ij} = p_{i\cdot} \cdot p_{\cdot j}$

连续型等价条件：$f(x,y) = f_X(x)f_Y(y)$

**考研要点**：二维随机变量的联合分布、边缘分布、条件分布及独立性判断是必考内容。常考题型包括：由联合密度求边缘密度和条件密度、判断独立性、求概率 $P((X,Y) \in D)$。

### 3.4 随机变量函数的分布

**$Z = X + Y$ 的密度**（卷积公式）：

$$f_Z(z) = \int_{-\infty}^{+\infty} f(x, z-x)dx = \int_{-\infty}^{+\infty} f(z-y, y)dy$$

若 $X$ 与 $Y$ 独立：$f_Z(z) = \int_{-\infty}^{+\infty} f_X(x)f_Y(z-x)dx$

**$Z = \max(X,Y)$ 的分布**：$F_Z(z) = F_X(z)F_Y(z)$（$X,Y$ 独立时）

**$Z = \min(X,Y)$ 的分布**：$F_Z(z) = 1 - [1 - F_X(z)][1 - F_Y(z)]$（$X,Y$ 独立时）

## 四、数字特征

### 4.1 数学期望

**离散型**：$E(X) = \sum_k x_k p_k$

**连续型**：$E(X) = \int_{-\infty}^{+\infty} x f(x)dx$

**随机变量函数的期望**：

$E(g(X)) = \sum_k g(x_k)p_k$（离散型）

$E(g(X)) = \int_{-\infty}^{+\infty} g(x)f(x)dx$（连续型）

$E(g(X,Y)) = \int_{-\infty}^{+\infty}\int_{-\infty}^{+\infty} g(x,y)f(x,y)dxdy$

**期望的性质**：
- $E(C) = C$
- $E(CX) = CE(X)$
- $E(X + Y) = E(X) + E(Y)$
- 若 $X$ 与 $Y$ 独立，则 $E(XY) = E(X)E(Y)$

### 4.2 方差

$$D(X) = E[(X - E(X))^2] = E(X^2) - [E(X)]^2$$

**方差的性质**：
- $D(C) = 0$
- $D(CX) = C^2 D(X)$
- $D(X + C) = D(X)$
- $D(X \pm Y) = D(X) + D(Y) \pm 2\text{Cov}(X,Y)$
- 若 $X$ 与 $Y$ 独立，则 $D(X \pm Y) = D(X) + D(Y)$

**切比雪夫不等式**：$P(|X - E(X)| \geq \varepsilon) \leq \frac{D(X)}{\varepsilon^2}$

### 4.3 协方差与相关系数

$$\text{Cov}(X,Y) = E[(X - E(X))(Y - E(Y))] = E(XY) - E(X)E(Y)$$

$$\rho_{XY} = \frac{\text{Cov}(X,Y)}{\sqrt{D(X)D(Y)}}$$

**相关系数的性质**：
- $|\rho_{XY}| \leq 1$
- $|\rho_{XY}| = 1 \Leftrightarrow$ $X$ 与 $Y$ 以概率 1 线性相关
- $\rho_{XY} = 0$ 称 $X$ 与 $Y$ 不相关
- 独立 $\Rightarrow$ 不相关，不相关 $\not\Rightarrow$ 独立
- 对二维正态分布，不相关 $\Leftrightarrow$ 独立

**考研要点**：期望和方差的计算是基础，协方差和相关系数是常考内容。需注意独立与不相关的关系，以及二维正态分布的特殊性质。

## 五、大数定律与中心极限定理

### 5.1 大数定律

**切比雪夫大数定律**：设 $X_1, X_2, \cdots$ 相互独立，方差有共同上界，则 $\frac{1}{n}\sum_{i=1}^{n}X_i \xrightarrow{P} \frac{1}{n}\sum_{i=1}^{n}E(X_i)$

**辛钦大数定律**：设 $X_1, X_2, \cdots$ 独立同分布，且 $E(X_i) = \mu$ 存在，则 $\frac{1}{n}\sum_{i=1}^{n}X_i \xrightarrow{P} \mu$

**伯努利大数定律**：设 $n_A$ 为 $n$ 次独立试验中事件 $A$ 发生的次数，$p$ 为每次试验中 $A$ 发生的概率，则 $\frac{n_A}{n} \xrightarrow{P} p$

### 5.2 中心极限定理

**列维-林德伯格定理（独立同分布中心极限定理）**：

设 $X_1, X_2, \cdots$ 独立同分布，$E(X_i) = \mu$，$D(X_i) = \sigma^2 > 0$，则：

$$\frac{\sum_{i=1}^{n}X_i - n\mu}{\sigma\sqrt{n}} \xrightarrow{d} N(0,1)$$

**棣莫弗-拉普拉斯定理**：

设 $X \sim B(n,p)$，则当 $n$ 充分大时：

$$P(a \leq X \leq b) \approx \Phi\left(\frac{b - np}{\sqrt{np(1-p)}}\right) - \Phi\left(\frac{a - np}{\sqrt{np(1-p)}}\right)$$

**考研要点**：中心极限定理常用于近似计算，关键是将问题转化为标准正态分布。大数定律的证明一般不考，但需理解其含义和应用。

## 六、参数估计

### 6.1 矩估计

用样本矩代替总体矩来估计参数的方法。

设总体 $X$ 的分布中含有 $k$ 个未知参数 $\theta_1, \theta_2, \cdots, \theta_k$，令：

$$E(X^j) = \frac{1}{n}\sum_{i=1}^{n}X_i^j, \quad j = 1, 2, \cdots, k$$

解此方程组得矩估计量 $\hat{\theta}_1, \hat{\theta}_2, \cdots, \hat{\theta}_k$。

### 6.2 最大似然估计

设总体 $X$ 的概率密度（或分布律）为 $f(x;\theta)$，样本 $X_1, X_2, \cdots, X_n$ 的似然函数为：

$$L(\theta) = \prod_{i=1}^{n} f(x_i;\theta)$$

取对数：$\ln L(\theta) = \sum_{i=1}^{n} \ln f(x_i;\theta)$

令 $\frac{d\ln L(\theta)}{d\theta} = 0$（多参数时对各参数求偏导为零），解得最大似然估计。

**注意**：当似然方程无解或导数不存在时，需用定义直接求 $L(\theta)$ 的最大值点。例如均匀分布 $U(0,\theta)$ 的最大似然估计为 $\hat{\theta} = \max(X_1, X_2, \cdots, X_n)$。

### 6.3 估计量的评价标准

**无偏性**：$E(\hat{\theta}) = \theta$

**有效性**：在无偏估计中，方差越小越有效

**一致性（相合性）**：$\hat{\theta} \xrightarrow{P} \theta$（$n \to \infty$）

**重要结论**：
- 样本均值 $\bar{X}$ 是总体均值 $\mu$ 的无偏估计
- 样本方差 $S^2 = \frac{1}{n-1}\sum_{i=1}^{n}(X_i - \bar{X})^2$ 是总体方差 $\sigma^2$ 的无偏估计
- $S^2$ 的分母是 $n-1$ 而非 $n$，这是保证无偏性的关键

## 七、假设检验

### 7.1 假设检验的基本思想

基于小概率原理：在一次试验中，小概率事件几乎不可能发生。若在原假设 $H_0$ 成立的条件下，观测到的事件概率很小（小于显著性水平 $\alpha$），则拒绝 $H_0$。

### 7.2 两类错误

- **第一类错误（弃真）**：$H_0$ 为真时拒绝 $H_0$，概率为 $\alpha$
- **第二类错误（取伪）**：$H_0$ 为假时接受 $H_0$，概率为 $\beta$
- 当样本量固定时，减小 $\alpha$ 则增大 $\beta$，反之亦然

### 7.3 正态总体参数的假设检验

**单个正态总体**：

| 原假设 | 检验统计量 | 拒绝域 |
|---|---|---|
| $H_0: \mu = \mu_0$（$\sigma^2$ 已知） | $Z = \frac{\bar{X} - \mu_0}{\sigma/\sqrt{n}}$ | $|Z| > z_{\alpha/2}$ |
| $H_0: \mu = \mu_0$（$\sigma^2$ 未知） | $t = \frac{\bar{X} - \mu_0}{S/\sqrt{n}}$ | $|t| > t_{\alpha/2}(n-1)$ |
| $H_0: \sigma^2 = \sigma_0^2$ | $\chi^2 = \frac{(n-1)S^2}{\sigma_0^2}$ | $\chi^2 > \chi^2_{\alpha/2}(n-1)$ 或 $\chi^2 < \chi^2_{1-\alpha/2}(n-1)$ |

**考研要点**：假设检验在考研中要求相对较低，主要掌握基本思想和单个正态总体的检验方法。需注意 $Z$ 检验与 $t$ 检验的选择条件（$\sigma^2$ 已知用 $Z$ 检验，$\sigma^2$ 未知用 $t$ 检验），以及 $\chi^2$ 检验用于方差的假设检验。
